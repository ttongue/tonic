This set of forms will guide you through the organizational membership signup process for the Tech Valley Center of Gravity. Organizational memberships are purchased for a specific number of licenses which determines how many simultaneous users can be in the facility. All organizational memberships share the following features:

  • Each user can have their own membership card
  • Each user is eligible for equipment training classes
  • Each license purchased may be used by any (safety trained) user of the purchasing organization.
  • Access to the makerspace 24/7
  • Active users may bring guests under conditions common to all guests (no unsupervised machine/tool usage, signed guest waivers, etc.)

Please select the type of membership plan you wish to enroll in:
    Organization Membership - 1 license - $200/month
      • As many users of organization as desired can be approved to use a single license

    Organization Membership - 2 license - $400/month
      • As many users of organization as desired can be approved to use the 2 licenses

    Organization Membership - 3 license - $600/month
      • As many users of organization as desired can be approved to use the 3 licenses

    Organization Membership - 4 licenses - $800/month
      • As many users of organization as desired can be approved to use the 4 licenses
